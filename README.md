# Privacy-Enhanced Anonymous Cloud Authentication Framework

## 📌 Project Overview

This project proposes a secure and privacy-preserving cloud authentication system that enables users to access cloud services anonymously while ensuring strong security using multi-factor authentication and encryption mechanisms.

The system prevents identity leakage, protects sensitive data, and enhances trust in cloud environments.

---

## 🚀 Technologies Used

- Python
- Django Framework
- MySQL Database
- ECC Encryption
- ChaCha20 Encryption
- HTML / CSS / JavaScript

---

## 🔐 Key Features

- Anonymous user authentication
- Multi-factor authentication protection
- Secure file upload and storage
- End-to-end encryption using ECC
- Fast symmetric encryption using ChaCha20
- User session security
- Cloud privacy protection

---

## ⚙️ System Modules

- User Registration & Login
- Authentication Token Generation
- Encryption & Decryption Module
- File Upload Module
- Secure Cloud Access

---

## ▶️ How to Run Project

1. Clone repository

git clone <repo-link>

2. Install dependencies

pip install -r requirements.txt

3. Run server

python manage.py runserver
